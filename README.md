# VAULT

A minimal black-on-black music library web app.

## Run locally
Just open `index.html` in your browser.

If your browser blocks local audio loading, run a tiny server:
- Python:
  - `python -m http.server 8080`
  - open http://localhost:8080

## Deploy
### GitHub Pages
1. Create a repo and push these files
2. Settings → Pages → Deploy from branch → main → /root
3. Your site goes live

## Add your music
Edit `tracks.js` and replace `src` with real MP3 URLs.
